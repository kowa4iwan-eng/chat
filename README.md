# Тестовий сайт

Це простий тестовий сайт, створений через ChatGPT у GitHub.

## Файли

- `index.html` — головна сторінка
- `style.css` — дизайн сайту
- `script.js` — маленька інтерактивність

## Як включити сайт через GitHub Pages

1. Відкрий репозиторій на GitHub.
2. Зайди в **Settings**.
3. В меню зліва відкрий **Pages**.
4. У полі **Source** вибери **Deploy from a branch**.
5. Branch: `main`, Folder: `/root`.
6. Натисни **Save**.

Після цього сайт має бути доступний приблизно за адресою:

`https://kowa4iwan-eng.github.io/chat/`
