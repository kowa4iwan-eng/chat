# Shadow Chat — демо секретного месенджера

Це перша тестова версія закритого веб-месенджера, створена через ChatGPT у GitHub.

## Що вже є

- Вхід по логіну і паролю
- Адмін-панель
- Створення користувачів
- Блокування / активація користувачів
- Виставлення прав при створенні користувача
- Створення груп і додавання учасників
- Приватні чати між користувачами
- Групові чати
- Онлайн-статус
- Смайли
- Відправка фото і відео як демо-прев’ю
- Повна адаптація під телефон
- Збереження демо-даних у браузері через `localStorage`

## Тестовий вхід

- Адмін: `admin` / `admin123`
- Користувач: `ivan` / `1111`
- Користувач: `natasha` / `2222`
- Користувач: `manager` / `3333`

## Важливо про безпеку

Це поки **демо-прототип для GitHub Pages**. Він показує дизайн і логіку роботи, але це ще не справжній секретний месенджер.

Для реальної секретної переписки потрібно додати:

- backend-сервер
- базу даних
- WebSocket для живих повідомлень
- справжню авторизацію
- хешування паролів
- права доступу на сервері
- шифрування повідомлень
- захист файлів і медіа

## Як включити сайт через GitHub Pages

1. Відкрий репозиторій на GitHub.
2. Зайди в **Settings**.
3. В меню зліва відкрий **Pages**.
4. У полі **Source** вибери **Deploy from a branch**.
5. Branch: `main`, Folder: `/root`.
6. Натисни **Save**.

Після цього сайт має бути доступний приблизно за адресою:

`https://kowa4iwan-eng.github.io/chat/`
