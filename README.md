# Shadow Chat — realtime messenger

Це Node.js застосунок з Express і Socket.IO. На відміну від попередньої HTML-версії, повідомлення тепер проходять через сервер і можуть доходити між різними пристроями.

## Що вже є

- логін і пароль;
- серверна сесія після входу;
- адмін-панель;
- створення користувачів;
- блокування та активація користувачів;
- створення груп;
- приватні чати;
- групові чати;
- онлайн-статус;
- смайли;
- фото і відео як прев’ю;
- мобільна адаптація;
- повідомлення зберігаються у `data/db.json`;
- нові повідомлення приходять через Socket.IO.

## Тестовий вхід

- `admin` / `admin123`
- `ivan` / `1111`
- `natasha` / `2222`
- `manager` / `3333`

## Запуск локально

Потрібен Node.js 18 або новіший.

```bash
npm install
npm start
```

Після запуску відкрий:

```text
http://localhost:3000
```

## Перевірка повідомлень

1. Відкрий сайт у двох браузерах або на телефоні й комп’ютері.
2. В одному зайди як `admin`.
3. В іншому зайди як `ivan`.
4. Напиши повідомлення в приватному чаті.
5. Воно має з’явитися одразу на другому пристрої.

## Важливо

GitHub Pages для цієї версії не підходить, бо він не запускає Node.js сервер. Для роботи онлайн потрібен VPS або Node.js hosting.

## Основні файли

- `server.js` — сервер, API і Socket.IO;
- `package.json` — залежності та запуск;
- `index.html` — сторінка;
- `style.css` — дизайн;
- `script.js` — frontend;
- `data/db.json` — створюється автоматично після першого запуску.
